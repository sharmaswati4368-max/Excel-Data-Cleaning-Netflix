# Excel-Data-Cleaning-Netflix
# Excel Data Cleaning & Formatting Task

## Dataset
Netflix Movies and TV Shows dataset (Kaggle)

## Tools Used
- Microsoft Excel

## Cleaning Steps
- Identified missing values using filters and conditional formatting
- Removed duplicates after creating a backup sheet
- Validated date and numeric formats
- Created a separate Cleaned_Data sheet
- Added Data_Quality_Notes to document data issues

## Output Files
- Raw_Data.csv
- cleaned_dataset.csv

## Final Outcome
A clean and structured dataset ready for analysis.
